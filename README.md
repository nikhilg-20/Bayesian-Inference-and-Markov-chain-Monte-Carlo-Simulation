# **Bayesian Inference and Regression Lab**

This repository contains my work for a **Bayesian Inference and Regression Lab** completed as part of my Machine Learning / Probabilistic Modeling coursework.

The objective of this lab is to understand how **Bayesian methods estimate model parameters, analyze uncertainty, and update beliefs as more data becomes available.**

---

# **Project Overview**

This project explores Bayesian inference concepts through practical experiments and visualizations.

Key topics covered in this lab include:

- Understanding **probability density functions and Bayesian posterior distributions**
- Observing how **parameter estimates change as more data is added**
- Visualizing **density contours using equipotential plots**
- Studying variation in **regression lines generated from posterior samples**
- Interpreting **uncertainty in model predictions**

The lab demonstrates how **Bayesian learning gradually reduces uncertainty as the amount of observed data increases.**

---

# **Dataset**

The project uses the **Olympic 100-meter race dataset**, which contains historical medal-winning times from the Olympic Games.

This dataset is used to study regression modeling and understand how **Bayesian inference estimates the relationship between year and winning time.**

---

# **Methodology**

The following steps were performed in this lab:

1. Load the dataset containing Olympic race results  
2. Apply **Bayesian regression techniques** to estimate model parameters  
3. Generate **contour plots representing probability density functions**  
4. Draw **regression lines sampled from the posterior distribution**  
5. Observe how these lines change as more data is incorporated  
6. Analyze the final posterior distribution to understand **parameter certainty**

---

# **Key Observations**

- Early in the experiment, the model shows **high uncertainty**, producing many different regression lines.
- As more data is added, the **density contours become tighter**, indicating stronger confidence in parameter estimates.
- The variation between regression lines decreases as the **posterior distribution becomes more concentrated**.
- The final density plot represents the **most likely parameter region after considering all observed data.**
